# Electric Point

Electric Point is a React ecommerce website for an electrical components store in Patna.

## Pages

- Home
- Shop
- Product detail
- Cart
- Checkout
- My Account
- Contact
- Blog

## Scripts

```bash
npm start
npm run build
```

## Design Notes

The site uses `src/styles/base.css` as the shared design system for colors, spacing, typography, responsive layouts, image rounding, forms, product grids, footer, and page sections.
